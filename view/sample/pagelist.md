# ページリスト

## 1. 公開ページ

### トップページ (/)

- **概要**: アプリケーションのランディングページ。未ログインユーザーもアクセス可能。
- **機能**:
  - アプリケーションの概要説明
  - 公開されている議論（Discussion）のピックアップ表示
  - ログイン / ユーザー登録画面への遷移

## 2. ユーザー機能

### ダッシュボード (/dashboard)

- **概要**: ログイン後のホーム画面。
- **機能**:
  - 自分のプロジェクト一覧の概略表示
  - 最近更新された議論の表示
  - 新規プロジェクト・議論作成へのショートカット

### プロジェクト一覧 (/projects)

- **概要**: ユーザーが管理するプロジェクトの一覧。
- **関連データ**: `ProjectSchema`
- **機能**:
  - プロジェクトの新規作成
  - プロジェクトの一覧表示

### プロジェクト詳細 (/projects/:id)

- **概要**: 特定のプロジェクトの詳細と、それに紐づく議論の管理。
- **機能**:
  - プロジェクト名の編集
  - プロジェクトに関連付けられた議論の一覧表示
  - 議論の新規作成（このプロジェクトに紐づく）
  - 既存の議論をこのプロジェクトに追加
  - プロジェクトの削除

## 3. 議論（Discussion）機能

### 議論一覧 (/discussions)

- **概要**: 全体の議論一覧。
- **関連データ**: `DiscussionSchema`
- **機能**:
  - 公開レベル (`VisibilityLevel`) に基づく議論の検索・一覧表示
  - ステータス (`open`, `closed`, `archived`) によるフィルタリング

### 議論作成 (/discussions/new)

- **概要**: 新しい議論を開始する。
- **機能**:
  - テーマ (`theme`)、背景 (`background`) の入力
  - 適用するルール (`ruleId`) の選択
  - 公開レベル (`visibilityLevel`) の設定
  - コメント許可レベル (`commentPermissionLevel`) の設定

### 議論詳細・ボード (/discussions/:id)

- **概要**: 議論を行うメイン画面。ツリー構造でコメントを表示。
- **関連データ**: `DiscussionSchema`, `CommentSchema`, `NoteSchema`, `IssueSchema`
- **機能**:
  - **議論ビュー**:
    - テーマ、背景、現在の結論 (`conclusion`) の表示
    - コメントツリーの可視化（`CommentType`ごとの色分け表示）
  - **アクション**:
    - コメントの投稿（許可レベルに基づく）
      - 親コメントの選択
      - コメント種類 (`commentTypeId`) の選択（ルールに基づく）
    - ノート (`Note`) の作成・閲覧（自分用メモ）
    - ノートからコメントへの変換
    - 指摘 (`Issue`) の作成（矛盾や循環論法の指摘）
  - **管理（オーナー向け）**:
    - 結論の更新
    - ステータスの変更（クローズ、アーカイブ）
    - 設定編集画面への遷移

### 議論設定 (/discussions/:id/settings)

- **概要**: 既存の議論の設定変更。
- **機能**:
  - テーマ、背景の修正
  - 公開レベル、コメント許可レベルの変更

## 4. ルール管理

### ルール一覧 (/rules)

- **概要**: 議論で使用するルールの管理。
- **関連データ**: `RuleSchema`
- **機能**:
  - ルールの新規作成
  - 作成したルールの一覧表示

### ルール詳細・エディタ (/rules/:id)

- **概要**: ルールの構造（コメントの種類と遷移）を定義。
- **関連データ**: `CommentTypeSchema`, `CommentTypePathSchema`
- **機能**:
  - ルール名、説明の編集
  - **コメント種類定義**:
    - 新しいコメント種類の追加（名前、説明、色）
  - **パス定義**:
    - コメント種類間の許可された遷移（From -> To）の設定
